CSS (Cascading Style Sheets)

CSS es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en un lenguaje de marcado como HTML. Es fundamental en el desarrollo web para controlar el diseño, la apariencia visual y la disposición de los elementos en una página web.

JavaScript

JavaScript es un lenguaje de programación esencial para la web moderna. Permite añadir interactividad, dinamismo y funcionalidades complejas a las páginas web, mejorando la experiencia del usuario. Desde validar formularios hasta crear animaciones y aplicaciones web completas, JavaScript juega un papel crucial.

Tabla de Atributos CSS

Atributo Descripción Valores Posibles Ejemplo de Uso Fuente
font-size Define el tamaño de la fuente del texto. <length> (px, em, rem), <percentage>, palabras clave (small, medium, large), etc. 
p {\n  font-size: 16px;\n}
 MDN
border-radius Permite redondear las esquinas de un elemento. <length> (px, em), <percentage>. Se pueden especificar hasta 4 valores para cada esquina. 
div {\n  border-radius: 10px;\n}
 MDN
display Especifica el tipo de caja de renderizado usada por un elemento (bloque, en línea, flex, grid, etc.). block, inline, inline-block, flex, grid, none, etc. 
span {\n  display: block;\n}
 MDN
display: flex Activa el contexto de formato Flexbox para los hijos directos del contenedor, permitiendo un control avanzado sobre la alineación y distribución del espacio. El valor es flex. Se complementa con propiedades como flex-direction, justify-content, align-items, etc. 
.container {\n  display: flex;\n  justify-content: space-between;\n}
 MDN
position Especifica el método de posicionamiento utilizado para un elemento (static, relative, absolute, fixed, sticky). static, relative, absolute, fixed, sticky. 
.tooltip {\n  position: absolute;\n  top: 0;\n  left: 20px;\n}
 MDN
box-shadow Añade efectos de sombra alrededor del marco de un elemento. Se pueden especificar múltiples sombras. none | <shadow> [ , <shadow> ]* donde <shadow> es inset? && <length>{2,4} && <color>? 
div {\n  box-shadow: 5px 5px 15px rgba(0,0,0,0.3);\n}
 MDN
overflow Controla cómo se muestra el contenido que excede las dimensiones de su contenedor. visible, hidden, clip, scroll, auto. 
.text-box {\n  height: 100px;\n  overflow: auto;\n}
 MDN
z-index Especifica el orden de apilamiento (profundidad) de un elemento posicionado. Elementos con mayor z-index se muestran sobre elementos con menor z-index. auto | <integer>. 
.modal {\n  position: fixed;\n  z-index: 1000;\n}
 MDN
opacity Define el nivel de opacidad de un elemento. 0 es totalmente transparente, 1 es totalmente opaco. <number> entre 0.0 y 1.0. 
.overlay {\n  opacity: 0.8;\n}
 MDN
margin Establece el espacio de margen en los cuatro lados de un elemento. Es el espacio fuera del borde. <length> (px, em, rem), <percentage>, auto. Se pueden especificar 1, 2, 3 o 4 valores. 
p {\n  margin: 10px 20px;\n}
 MDN
padding Establece el espacio de relleno en los cuatro lados de un elemento. Es el espacio entre el contenido y el borde. <length> (px, em, rem), <percentage>. Se pueden especificar 1, 2, 3 o 4 valores. 
div {\n  padding: 15px;\n}
 MDN
color Establece el color del texto de un elemento. <color> (nombre, hex, rgb, hsl). 
h1 {\n  color: #3498db;\n}
 MDN
background-color Establece el color de fondo de un elemento. <color> (nombre, hex, rgb, hsl), transparent. 
body {\n  background-color: #ecf0f1;\n}
 MDN
width Especifica el ancho del área de contenido de un elemento. auto, <length> (px, em, rem), <percentage>, max-content, min-content, fit-content(). 
.container {\n  width: 80%;\n}
